# HG Game - Gaming App 

A one-page Next.js (App Router) + TailwindCSS project

- Sidebar
- Header
- Dashboard-style main area
- Latest Bets & Race section
- Sign-in and Sign-up **modals** via hash routes (`/#signin`, `/#signup`)

## Tech

- Next.js (v15 compatible) - App Router
- React 18
- Tailwind CSS 3
